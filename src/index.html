Popover компонент

При инициализации экземпяра popover-компонента нужно передать три параметра: элемент DOM дерева, по клику на который будет появляться popover, заголовок и текст.

Скрытие popover-компонента происходит по повторному нажатию на элемент DOM дерева или на кнопку ОК внутри popover-компонента.

Для демонстрации нажмите на кнопки под этим текстом.
